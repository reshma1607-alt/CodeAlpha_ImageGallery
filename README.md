# 📸 CodeAlpha Image Gallery

## 📌 Project Description
A responsive Image Gallery built using HTML, CSS, and JavaScript. The gallery allows users to view images in a modern layout with interactive features.

## ✨ Features
- Responsive Image Gallery
- Hover Effects
- Lightbox View
- Previous & Next Navigation
- Clean and Modern UI

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
